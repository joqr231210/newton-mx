1. Objeto

Newton es una plataforma de software como servicio (SaaS) diseñada para que empresas de servicios especializados realicen costeo de proyectos, generación de cotizaciones en PDF, seguimiento de aprobaciones y gestión de órdenes de trabajo. Estos Términos regulan el acceso y uso de dicha plataforma, incluyendo su sitio web, aplicación web y aplicación móvil.

Newton se reserva el derecho de suspender o cancelar cuentas que incumplan estos Términos o que estén asociadas a actividades fraudulentas.

El usuario se compromete a utilizar la plataforma únicamente para fines lícitos y relacionados con la gestión de su negocio. Queda prohibido:
